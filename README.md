### Installation

To install this package you need package `devtools`:

```{r eval=F}
install.packages("devtools")
```

Then type:

```{r eval=F}
devtools::install_github("reyzaguirre/rhep")
```

### Usage

To load the package type:

```{r eval=F}
library(rhep)
```

To see the documentation type:

```{r eval=F}
help(package = rhep)
```
